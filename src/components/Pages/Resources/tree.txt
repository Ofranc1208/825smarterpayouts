MAINTENANCE NOTES
================================================================================

To Add New Insurance Companies:
   1. Edit: src/components/Pages/Resources/types.ts
   2. Add new company object to companies array
   3. Include all required fields (name, phone, email, etc.)
   4. Add former names if applicable
   5. Specify document requirements in acblsadapos array
   6. Add notes if needed (e.g., special instructions)

To Update Acronyms:
   1. Edit: src/components/Pages/Resources/types.ts
   2. Add new acronym to acronymDescriptions object
   3. The legend will automatically update

To Modify Styling:
   1. All styles use design system tokens
   2. Update tokens in: src/components/shared/styles/
   3. Changes will apply site-wide automatically
